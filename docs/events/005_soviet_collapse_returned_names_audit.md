# Event 005 Returned Names Restoration Audit

Audit date: 2026-05-21

This audit covers the ancient and medieval restoration requirement from the final clean merged Part 6 specification. The implementation now has two layers: dynamic host-country Returned Names decisions for existing eligible countries, and standalone high-chaos ancient restoration tags for the Khazar, Sogdian, Khwarazmian, and Alan claim lines. The standalone tags have country/history files, flags, localisation, focused spawn/setup logic, event notices, event-log stages, League interaction, and separate per-tag restoration focus trees.

## Remaining Deviation

The Returned Names system satisfies the standalone-tag surface at source level: each claim line has a tag, country/history files, flags, localisation, focus content, release/spawn logic, report event, and event-log integration. The former shared-tree blocker is resolved: `KZR`, `SOG`, `KHW`, and `ALN` each load a separate 15-focus restoration tree with political, industry, guard/military, diplomacy, claim, symbolic, and expansionist payoffs. The remaining deviation is live validation: these source checks do not claim an in-game playthrough.
