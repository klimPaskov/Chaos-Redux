# Event 005 Ukraine Bread State Decisions

Ukraine's high-chaos Bread State branch now exposes route-gated decisions through the breakaway decision category. The decisions use the existing focus flags from `ukr_soviet_collapse_black_soil_oath`, `ukr_soviet_collapse_grain_census_of_everyone`, `ukr_soviet_collapse_no_one_leaves_the_bread_line`, and `ukr_soviet_collapse_last_harvest_plan`.

The route tracks two country variables:

- `soviet_collapse_ukraine_grain_authority`: how much coercive grain administration Ukraine can turn into logistics and emergency production.
- `soviet_collapse_ukraine_bread_state_coercion`: how much strain the ration-line state is imposing on villages.

The decision loop adds real tradeoffs. Black-soil committees and the grain census build authority, depots, fortifications, rail capacity, and League weight, but they raise coercion and push Soviet old-movement pressure. Locking the bread line is stronger in wartime and creates local defenses, manpower, and war support at the cost of stability. Village ration relief spends capacity to reduce coercion. The Last Harvest Plan requires enough grain authority and converts the route into emergency arms production while further stressing the Soviet collapse ledger.

No new icons are required in this chunk. The decisions reuse existing Event 005 decision icons so gameplay can be wired before the lower-priority icon pass.
