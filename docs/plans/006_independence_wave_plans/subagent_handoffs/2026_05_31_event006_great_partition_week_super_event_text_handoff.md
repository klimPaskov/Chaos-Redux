## Recommended Package

### Title

`The Great Partition Week`

Reason:

- clearest and strongest at a glance
- sounds administrative and historical rather than lyrical
- communicates scale without implying one final collapse
- fits a super-event built around count shock, repeatability, and many simultaneous recognitions

### Selected Quote

`"Self-determination" is not a mere phrase. It is an imperative principle of action, which statesmen will henceforth ignore at their peril.`

- Speaker: Woodrow Wilson
- Source: `Address to Congress on International Order`
- Date: February 11, 1918
- Source URL: https://www.presidency.ucsb.edu/documents/address-congress-international-order
- Attribution confidence: High
- Copyright note: historical U.S. presidential speech text; underlying text is public domain in the United States

Why it fits:

- It captures the exact Event 006 escalation point: what had looked like rhetoric or isolated precedent has become a repeatable governing principle.
- It is about statesmen losing the option to treat border change as a contained exception.
- It fits bureaucratic panic better than a purely martial frontier quote.
- It does not imply that one empire has simply shattered; it implies that the political language of release now acts on its own.

### Recommended Button Line

`Count them again.`

- Type: original line from the Event 006 super-event prompt
- Copyright risk: none
- Reason: best match for the role because it sounds like a foreign office, map room, or wire desk reacting to an unstable state count rather than making a grand ideological statement

### Description Direction

Keep the final description procedural and unsettling, not lyrical and not apocalyptic.

Recommended emphasis:

- telegraph offices, ministries, consulates, and newspapers cannot agree how many states now exist
- border commissions, recognition files, railway authorities, port offices, and provincial committees are all producing new realities at once
- at least some hosts still stand, which makes the wave feel less like terminal collapse and more like a contagious administrative method
- the world shock is that partition has become repeatable language, not a single exceptional crisis

Useful drafting direction for the main agent:

The first paragraph should establish the counting failure. The second should show many institutions trying to classify, certify, or deny the new states at the same time. The third should land on the real meaning: the map has become procedurally unstable, and governments now fear the method as much as any one release.

## Fit Explanation

This package fits the exact role because it treats the event as a crisis of administrative comprehension. The title marks the week as a named episode. The selected quote says the governing principle behind the wave can no longer be ignored as rhetoric. The button line reduces the reaction to the most practical panic possible: the count is no longer stable.

Just as important, this package does not read like Event 005. It does not center one imperial corpse, one capital, or one succession chain. It centers ministries, maps, recognition files, and a contagious release logic that can strike several hosts while leaving them alive.

The recommended text also leaves room for mixed packages. Historical-return states, city/free-ports, railways, local polities, and stranger releases can all fit under this presentation because the emphasis is on proliferating categories and failing administrative agreement.
