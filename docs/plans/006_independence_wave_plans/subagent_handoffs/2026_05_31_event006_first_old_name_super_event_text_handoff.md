# Event 006 Super-Event Text Handoff

## Scope

Text research only for Event 006 Independence Wave super-event candidate `independence_wave_first_old_name`.

Role:

- first Event 006 historical-return package becomes independent
- first global old-name reveal only
- Independence Wave origin only, not Event 005
- tone shift from modern separatism to old-state restoration language

No gameplay, localisation, GFX, sound, music, or spreadsheet files were edited.

## Recommended Package

### Title

`An Old Name Returns`

Reason:

- clearest at a glance
- immediately communicates the tonal pivot from ordinary breakaway politics to historical-return legitimacy
- serious and political without implying literal ancient restoration

### Selected Quote

`"Love your country. It is your name, your glory, your sign among the peoples."`

- Speaker: Giuseppe Mazzini
- Source: `To the Young Men of Italy`
- Occasion / date: address at Milan, July 25, 1848
- Source URL: https://en.wikisource.org/wiki/The_World%27s_Famous_Orations/Volume_7/To_the_Young_Men_of_Italy
- Attribution confidence: High
- Copyright note: original speech is public domain; the English translation used on Wikisource is from a pre-1931 collection and is public domain in the United States

Why it fits:

- It directly ties countryhood to a name recognized "among the peoples," which is exactly the reveal this super-event needs.
- It frames the moment as political legitimacy and recognition, not mystical resurrection.
- It works for multiple historical-return packages without naming one region too narrowly.
- It sounds formal and nineteenth-century enough for archive, treaty-table, and diplomatic-wire presentation.

### Recommended Button Line

`The map remembers.`

- Type: original line
- Copyright risk: none
- Reason: short, uncanny, and political; it suggests remembered legitimacy without claiming the past has literally returned unchanged

### Description Direction

Foreign offices expected another minor separatist report and instead receive an older state name in the telegrams, dossiers, and recognition files. The description should emphasize that archives, local elites, councils, clergy, militias, or petitioners have made an old political vocabulary usable again. The key point is not that the past has come back intact, but that chaos has made an old claim newly legible and newly dangerous.

Recommended emphasis:

- diplomats are startled by the name before they understand the borders
- the new state argues legitimacy through memory, precedent, and local recognition
- outside powers cannot dismiss it as a routine municipal revolt anymore
- avoid supernatural language and avoid Soviet-collapse framing

## Fit Explanation

This package fits the exact role because the first historical-return release is not just another independence event. It is the first moment when Event 006 stops looking like modern fragmentation alone and starts producing states that justify themselves through remembered names, older claims, and revived legitimacy.

The recommended title is plain enough to be legible in-game. The Mazzini quote makes the name itself the political fact. The button line keeps the tone restrained and uncanny. Together they signal that the world is not seeing a dead empire restored intact; it is seeing living actors use old state language to force recognition in the present.

Just as important, this package does not read like Event 005. There is no collapse-center framing, no post-Soviet republic language, and no implication that geography alone decides the presentation. The emphasis is on recognition shock, archival legitimacy, and the international meaning of an old name appearing on modern wires.

## Implementation Recommendation For Main Agent

- Use `An Old Name Returns` as the final title unless another Event 006 super-event already claims a near-identical phrasing.
- Use the Mazzini quote as the main `.q` line.
- Use `The map remembers.` as the `.a` line.
- Write the description as a diplomatic recognition shock: officials expected a small new state, then an old polity name arrives with seals, petitions, and borders attached.
- Keep the text clear that this is a modern political construction using historical memory, not a literal untouched state from the past.
