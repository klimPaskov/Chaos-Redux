# Event 6 Independence Wave Research Notes

These notes give the implementation agent a research baseline for high-chaos packages. They are not a substitute for final asset source review. Any real flag, symbol, leader, dynasty, or portrait still needs a source check during implementation.

## HOI4 release baseline

The event starts from ordinary HOI4-style release logic. A candidate can be an existing releasable, dead tag, generated tag, game-rule tag, decision tag, or formable-compatible tag if state ownership, cores, origin gating, and host survival checks pass.

Design use:

- baseline waves should prefer existing tags and cores
- generated or dormant tags should enter later
- high-chaos packages should not replace ordinary releasables too early
- every release must still protect the host's final state

## Ancient or old-state packages

### Assyria

Useful angle:

Assyria should be a high-chaos historical-return package tied to northern Mesopotamia and modern Assyrian identity. It should not imply that the ancient empire has simply returned unchanged. The route should split between modern civic Assyrian recognition, old-state restoration symbolism, patron protection, and high-chaos mythic escalation.

Playable hooks:

- archive petitions
- diaspora recognition mission
- old seal and modern council dispute
- protection against larger hosts
- claim ambition restrained by host survival

Asset caution:

Real Assyrian symbols and modern community symbols need review.

### Mesopotamia

Useful angle:

Mesopotamia is better as a river-federation or mandate-archive package than as a single ethnic state. It can use the political idea of the Tigris and Euphrates region, the legacy of Iraq's mandate-era state formation, and high-chaos border failure.

Playable hooks:

- river compact
- city councils
- archive of the mandate
- federation versus central ministry
- patron danger from outside powers

Asset caution:

Avoid treating all of Iraq or all ancient Mesopotamia as one simple identity.

### Volga Bulgaria

Useful angle:

Volga Bulgaria and Old Great Bulgaria can appear through Event 6, but only as Independence Wave historical-return packages. They should use Volga trade, archive, patron pressure, league politics, and old-state memory routes. They must not inherit Event 5 Soviet Collapse mechanics, focus trees, formables, event logs, or republic missions.

Playable hooks:

- Volga trade charter
- steppe and river compromise
- modern republic council
- old capital memory
- Event 6 origin check

Asset caution:

Shared tag assets need separate Event 6 documentation.

### Circassia and Mountain Republic

Useful angle:

These packages should focus on mountain congresses, exile memory, local defense, and small-state diplomacy. They fit high-chaos local-polity or historical-return logic in the Caucasus.

Playable hooks:

- mountain congress
- diaspora appeal
- militia defense
- protectorate danger
- border arbitration

Asset caution:

Source symbols carefully and avoid collapsing many peoples into one generic mountain identity.

### Bukhara, Khiva, Kokand

Useful angle:

Central Asian old-state packages can use emirate, khanate, oasis-city, and archive politics. They should not be ordinary early releases unless the mod already has safe tags.

Playable hooks:

- oasis administration
- emirate or khanate memory
- scholar council
- city trade route
- patron pressure from nearby powers

Asset caution:

Real dynastic and religious symbols require source review.

## Africa packages

### Asante

Useful angle:

Asante should be a local-polity or historical-return package based on state memory, court politics, and modern survival under high chaos. It should not become generic West African filler.

Playable hooks:

- court council
- stool symbolism after source review
- confederation route
- modern compromise route
- defense against patron control

### Sokoto

Useful angle:

Sokoto can be a high-chaos historical-return package with scholarly, administrative, and religious legitimacy routes. Use careful source review and avoid turning the package into caricature.

Playable hooks:

- scholar council
- provincial emirates
- archive claim
- reform versus restoration split
- patron and border caution

### Kanem-Bornu

Useful angle:

Kanem-Bornu fits the Lake Chad region and should use lake routes, caravan memory, court revival, and regional fragmentation.

Playable hooks:

- Lake Chad compact
- caravan tax route
- court restoration
- desert defense
- border arbitration

### Buganda

Useful angle:

Buganda can use kabaka court, lake-region politics, and protectorate dispute. It should have a modern compromise path and a monarchy-centered path if assets and leaders are safe.

Playable hooks:

- kabaka court
- lake trade
- protectorate records
- civic compromise
- patron leverage

### Barotseland

Useful angle:

Barotseland should be a local-polity package using Lozi political institutions, floodplain geography, and protectorate memory.

Playable hooks:

- floodplain council
- river boundary
- protectorate dispute
- community defense
- federation or autonomy settlement

### Zulu restoration

Useful angle:

Zulu should be a careful historical-return package with royal council, regimental memory, and modern state compromise. Avoid using it as generic warrior fantasy.

Playable hooks:

- royal council
- regimental memory
- civic charter
- border restraint
- host survival pressure

### Herero and Nama authorities

Useful angle:

These should use land testimony, survival memory, and local authority routes. Treat genocide memory soberly and avoid pulpy high-chaos framing unless a strange route is clearly separate.

Playable hooks:

- land testimony mission
- community defense
- memorial congress
- international observer route
- anti-patron route

## South America packages

### Mapuche Araucania

Useful angle:

Mapuche Araucania can be a local-polity or historical-return package centered on land congress, frontier treaties, community defense, and high-chaos recognition.

Playable hooks:

- land congress
- frontier treaty archive
- community guard
- confederation route
- border war danger

### Guarani

Useful angle:

Guarani can use river, language, mission, and cross-border community politics. It should not use generic jungle imagery.

Playable hooks:

- river communities
- language congress
- mission archives
- dispersed statehood problem
- recognition without patron

### Charrua

Useful angle:

Charrua should be a small, difficult local-polity package with memorial politics and sparse starting territory. It should usually be rare.

Playable hooks:

- memorial council
- minimal state release
- recognition struggle
- observer route
- high legitimacy challenge

### Aymara

Useful angle:

Aymara can use highland councils, cross-border identity, and federation mechanics. It should have a route that remains defensive and a route that pursues wider highland claims later.

Playable hooks:

- highland congress
- community militia
- language and land recognition
- cross-border federation
- claim ambition at high chaos

### Palmares

Useful angle:

Palmares is best as a high-chaos old-name and freedom-community package, not a direct modern state copy. It can connect to Quilombo memory, refuge, fortification, and liberation language.

Playable hooks:

- fortified settlement
- freedom charter
- hidden refuge network
- recognition crisis
- patron rejection

## Strange package research posture

The strange packages are invented, but they should use state language. Use ministries, laws, registries, censuses, border offices, rail timetables, archives, and citizenship categories. Avoid monster names unless the route specifically needs them.

Useful strange routes:

- necromantic custodianship
- anti-mankind directorate
- archive-state
- railway sovereignty gone impossible
- unmarked congress
- non-human border office

## Research follow-up needed during implementation

- verify which candidate tags already exist in Chaos Redux
- map state IDs and ownership checks for every package
- check all real flags and symbols
- check all leader and portrait candidates
- decide which packages are first-pass implementation and which remain planned
- make shared-tag origin tests, especially for Volga Bulgaria
- keep host survival logic above package ambition

## Research needs for formation routes

Formation routes need stronger research than ordinary releases because they claim a wider identity. Research should focus on institutions, symbols, old borders, treaties, rival claims, local authorities, and regional memory.

For each implemented historical-return or local-polity formation, prepare a short research note covering:

- historical anchor
- modern or period-relevant population anchor when relevant
- political institution or symbolic office
- possible capital or congress site
- symbols that must be sourced
- disputed claims or sensitive uncertainty
- route names and decision names that fit the region
- what should not be implied

Do not treat ancient or old-state names as if the ancient state simply returned unchanged. Event 006 should usually frame these as modern committees using old names, regional congresses rebuilding old institutions, local polities turning recognition into statehood, or high-chaos routes mythologizing the past.
