# Super-Event Prompt: Event 008 Tensions Rising

Research and design a super-event package for Event 008 `Tensions Rising`.

Use this prompt only for the first Stage IV firing of the `Diplomatic Fever` evolution track. Event 8 has no world-end scenario, this super-event is global escalation, not a terminal branch.

Read and follow:

- `AGENTS.md`
- `chaos-redux-super-events`
- `chaos-redux-event-assets` for image handoff
- Event 008 specs under `docs/specs/008_tensions_rising_specs/`

Do not edit localisation, event files, GFX, GUI, sound definitions, or spreadsheets. Produce research notes and handoff material.

## Role

First Stage IV reveal / global escalation. The world is not ending, but the old diplomatic line has disappeared. Every capital is acting as if someone else will make the first mistake.

## Working title

Primary: **The Red Line Disappears**  
Backup: **The Whole Map Holds Its Breath**

Avoid generic apocalypse titles.

## Description direction

The description should focus on lit ministries, sealed cables, quiet offices, officials denying the obvious, and the collapse of believable calm. It should not claim that war has started or that a world-end scenario has begun.

## Button/remark direction

Find a short, fitting remark or cultural allusion. It can be grim or bureaucratic. Examples of tone, not final sourced text:

- “No one admits it first.”
- “The lamps stay on.”
- “Every line moves now.”

If the remark uses a real cultural source, verify wording and keep copyright limits.

## Quote direction

Research several real quote candidates about fear, war before declaration, brittle peace, public denial, diplomacy, or crisis psychology. Prefer public-domain, historical, philosophical, literary, or political sources. Do not invent quotes or use unsourced quote-site copy.

Document for each candidate:

- exact quote excerpt
- speaker/author
- source work/speech/document
- year or period
- source URL
- attribution confidence
- copyright/public-domain note when known
- fit explanation

## Audio direction

Find a real licensed or public-domain musical track. It should feel tense, official, and nocturnal rather than battlefield-triumphant. Prefer 1–2 minutes, or document any exception and trim to appropriate length if allowed.

Reject unclear licenses, YouTube-only uploads without license, generated oscillator/test-tone tracks, drones, one-shot stingers, and placeholder provenance.

Document title, composer/creator, performer/source, source URL, license, duration, final `.ogg` suggestion, editing notes, and why it fits.

## Image direction

Coordinate with Event 008 asset prompt. Recommended image mode: generated super-event image.

Visual idea: a composite-feeling but period-authentic 1936–1945 night scene of embassy or ministry windows, telegraph/cable rooms, sealed dispatches, locked gates, and civil servants crossing dark courtyards. No readable generated text, no modern props, no battlefields, no generic map table as main subject.

Include selected quote, backup quotes, selected button text/remark, audio candidate recommendation, image handoff direction, uncertainty notes, and implementation notes for the main agent.
