# Super-event prompt for Event 007 Fury

Use `chaos-redux-super-events` for this package. Use the repository web research workflow from `AGENTS.md` for quote, cultural remark, and audio research. Do not invent quotes. Do not use music without verified source and license.

Event ID: `7`
Event slug: `fury`

## Super-event 1: Fury becomes a major

Role: escalation.

Trigger: the first active Fury actor becomes a major or crosses the Fury major threshold defined in the implementation.

Purpose: mark the point where Fury stops being a local war event and becomes a power that other countries must treat as a major threat.

Tone: cold, military, bureaucratic, and uneasy. The world does not yet know whether this is a new ideology, a state failure, or a country that learned to move too fast.

Title direction:

- short and specific.
- avoid generic world-ending language.
- possible directions: `A Minor No Longer`, `The Map Office Expands`, `The Small State Becomes Large`.

Description direction:

- explain that a country once treated as a small border actor has grown into a major power through rapid conquests.
- mention war offices, captured registries, and foreign observers.
- avoid revealing the world-end branch.

Button text direction:

- short, cold, and memorable.
- possible direction: `The map keeps moving`.
- If using a cultural reference, verify exact source and keep copyright-safe.

Quote direction:

- find a real quote about force becoming policy, borders changing through war, unchecked conquest, or administrative violence.
- Prefer public domain historical, literary, philosophical, or political sources.
- Keep quote short.

Image direction:

- generated super-event image.
- period-style war office in a once-small capital.
- map walls, border pins, dispatch clerks, officers, and files.
- no modern screens.
- no readable generated text.

Audio mood:

- tense march or orchestral public domain or clearly licensed track.
- 1 to 2 minutes preferred.
- not a drone, beep, test tone, or abstract ambience.
- document composition and recording rights separately.

## Super-event 2: The World in Fury

Role: world-end reveal.

Trigger: Fury terminal branch begins. Main Fury actor seeds other continents and creates or leads the main Fury faction.

Purpose: mark terminal state. Fury has become a world pattern.

Tone: final, global, administrative, and military. It should feel organized enough to be dangerous, not random noise.

Title direction:

- `The World in Fury` is the required scenario name and should be strongly considered for the super-event title.
- Alternative title can be used only if the scenario name remains visible elsewhere.

Description direction:

- explain that Fury offices and aligned states appear across continents.
- mention reports from several capitals.
- show that other continents now have Fury actors joining the main faction.
- make the terminal state clear without overexplaining mechanics.

Button text direction:

- short.
- possible direction: `Every border is a front`.

Quote direction:

- real quote about war spreading, order breaking, imitation of violence, or the world becoming a battlefield.
- Prefer public domain or historical source.
- Avoid unsourced internet quotes.

Image direction:

- generated super-event image.
- global map room with continent markers, radio operators, dispatch tables, and repeated small flags.
- period style.
- no modern screens.
- no readable text.

Audio mood:

- grave march, requiem-like orchestral piece, hymn, or public domain recording with clear rights.
- 1 to 2 minutes preferred.
- document source, license, final `.ogg`, sound id, and music table row.

## Optional super-event 3: Fury defeated after terminal branch

Role: defeat aftermath.

Trigger: all Fury actors are defeated after world-end Fury branch has been active long enough or after the Fury faction controlled enough land to reshape the campaign.

Purpose: show that the world survives, but the border system and war memory remain damaged.

Tone: reflective and tired. Avoid clean triumph.

Title direction:

- `The Quiet Borders`.
- `The Offices Close`.
- `After the March`.

Description direction:

- explain that Fury commands are dismantled, captured maps are archived, and reconstruction begins.
- mention remaining distrust and border militarization.

Button text direction:

- possible direction: `Count what remains`.

Quote direction:

- real quote about rebuilding after war, the cost of victory, or memory after violence.
- Prefer public domain source.

Image direction:

- generated or sourced if a suitable public domain archive image fits.
- abandoned border office, sealed maps, soldiers guarding an archive, tired officials.

Audio mood:

- slower public domain or clearly licensed reflective track.
- Not triumphant unless the chosen music still carries cost.

## Required documentation

For each selected super-event, create or update:

- title localisation.
- description localisation.
- button text localisation.
- quote localisation.
- quote research notes.
- image direction and asset handoff.
- audio candidate notes.
- selected audio with title, creator, performer or source, URL, license, duration, usage terms, final `.ogg`, sound id, and music table row.
- implementation notes for `global.current_super_event_audio_id` and settings-aware playback.

Do not claim a super-event package is complete with placeholder or undocumented audio.
