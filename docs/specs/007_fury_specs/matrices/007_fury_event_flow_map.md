# 007 Fury event flow map

```text
Eligible AI minor found
  apply Fury package
  load shared tree
  start weekly unit pulse
  pick weaker eligible neighbor
  declare war without warning
    if Fury loses
      cleanup and defeat log
    if target falls
      conquest settlement
      first-conquest news if first victory
      add compliance and overextension
      unlock coring decisions
      pick next target
        if target exists
          repeat war loop
        if no target exists
          no-neighbor branch
            if terminal eligibility
              world-end branch
            if no terminal eligibility
              contained Fury state
    if Fury becomes major
      Fury-major super-event
      world-threat source
      anti-Fury response
```
