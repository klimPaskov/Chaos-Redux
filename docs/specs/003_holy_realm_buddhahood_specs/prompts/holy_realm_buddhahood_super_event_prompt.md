# Super-Event Prompt: Holy Realm Buddhahood

Use `chaos-redux-super-events` to research and prepare the super-event packages for `holy_realm_buddhahood`.

Read:

- `specs/holy_realm_buddhahood_super_events.md`.
- `specs/holy_realm_buddhahood_spec_part_1.md`.
- `research/holy_realm_buddhahood_research_notes.md`.

Super-events:

1. `The Awakened One`, triggered when the Bodhisattva becomes the Buddha.
2. `Powers of the Awakened`, triggered when the first Buddha power is shown against a chaos country.
3. `The Final Silence`, triggered when the Final Silence ritual completes, with terminal and non-terminal variants.

For each super-event, provide:

- Final title, description, button text, and quote.
- Quote candidates from reliable sources, with exact wording and attribution confidence.
- Button text source notes if it uses a cultural, scriptural, or literary reference.
- Audio candidates, license verification, duration, source, usage terms, and final recommendation.
- Image direction handoff for the asset agent.
- Implementation notes for slot, audio id, and documentation.

Do not invent quotes. Do not use unclear audio. Do not leave placeholder audio for a completed super-event. Prefer public domain or clearly licensed sacred, chant, bell, or instrumental music that fits each role.
