# Bioweapon Operative Outbreak Operation

This mechanic adds state-targeted intelligence operations for covertly planting biological outbreak payloads.

## Player Flow

1. Build an intelligence network in the target country.
2. Select one of the plant-outbreak operations in the intelligence agency operation list.
3. Choose a valid controlled target state.
4. Assign operatives and required payload stockpiles.
5. On completion, the operation rolls a success chance based on disease infectiousness and network strength.
6. On success, the selected state receives the outbreak payload.
7. On failure, the payload is lost without starting an outbreak.

## Operations

- `operation_plant_plague_outbreak`
- `operation_plant_smallpox_outbreak`
- `operation_plant_weaponized_zombie_outbreak`

Plague and smallpox use fixed infectiousness constants in `bio_operation_plant_outbreak` because those pathogens do not have an upgrade path. Weaponized zombies use the creator country's `weaponized_zombie_infectiousness` value generated by the `weaponize_the_zombies` special project.

## Success Calculation

Success starts from `bio_operation_plant_outbreak.base_success`.

The operation then adds:

- pathogen infectiousness multiplied by `bio_operation_plant_outbreak.infectiousness_factor`
- a network-strength bonus at 40, 60, 80, or 100 network strength

The result is rounded and clamped between `bio_operation_plant_outbreak.min_success` and `bio_operation_plant_outbreak.max_success`.

## Effects

Plague and smallpox call the existing state contamination effects:

- `apply_plague_contamination`
- `apply_smallpox_contamination`

Weaponized zombies call the existing spawn path:

- `spawn_weaponized_zombie_outbreak_from_creator`

The operation saves short-lived event targets for the actor and victim:

- `bio_operation_actor`
- `bio_operation_victim`

These are regular event targets and only persist during the operation completion effect chain.

## Localisation

Operation and phase localisation is defined in `localisation/english/chaosx_operations_l_english.yml`.

The text stays abstract and gameplay-facing. It describes operatives, mission risk, state selection, and outbreak outcomes without procedural detail.
