# Event 005 Remaining Custom Flag Correction Handoff

No `.gfx` edits are needed for HOI4 country flags. The engine resolves these by tag and ideology/cosmetic flag filename from `gfx/flags/`, `gfx/flags/medium/`, and `gfx/flags/small/`.
