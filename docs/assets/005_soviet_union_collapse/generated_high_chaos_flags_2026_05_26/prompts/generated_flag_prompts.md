# Event 005 Generated High-Chaos Flag Prompts

Package: `generated_high_chaos_flags_2026_05_26`

Source mode: `$imagegen` built-in image generation.

These prompts were used to create first-batch fictional base-flag source art for Event 005 Soviet Collapse. Each prompt requested a clean horizontal flag source at HOI4-readable proportions, intentional alternate-history flag art, strong central composition, no readable text, no watermarks, no simple recolor treatment, and no upside-down orientation.

## SDZ

Fictional flag source art for the Security Directorate Zone: sealed archive state, cordon authority, prison-records directorate, all-seeing searchlight eye, locked file seal, barbed cordon line, charcoal gray, cold white, dark red seal accents, desaturated steel blue.

## TSC

Fictional flag source art for the Tunguska Star Committee: Siberian star-science authority, cosmic committee, impact zone, secret observatory, radiant meteor star, pine horizon, survey compass circle, midnight teal, aurora green, pale silver, small crimson warning accent.

## UDC
