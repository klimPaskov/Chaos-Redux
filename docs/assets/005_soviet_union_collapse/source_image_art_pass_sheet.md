# Event 005 Source Image Art Pass Sheet

Audit date: 2026-05-15.

This sheet tracks optional source/crop choices for the sourced Event 005 report and super-event images. All rows are already processed, converted, wired, and documented.

Optional art-pass choices for each row:

- Keep: current source and crop stay in place.
- Adjust: keep the same source, but adjust crop, contrast, grade, or framing.
- Replace: reject the source image and choose a different source.

Use these boxes only when a later art pass changes or confirms a source/crop choice. Empty boxes do not block Event 005 completion.
