# Event 006 Decision Icon Manifest

Status: processed, converted, and wired.

This package adds reusable static 32x32 decision icons for implemented Event 006 Independence Wave decisions. The source artwork is generated Event 006 icon art from the decision-category tranche, processed into smaller reusable decision icons. No country flag assets, country files, history files, or Event 005 assets are used.

## Processing Notes

- Source mode: generated source art, reused from the Event 006 decision-category icon source set.
- Parent processing: source PNGs were copied into this package, magenta-keyed to alpha, trimmed, resized, centered on 32x32 transparent canvases, and converted to DDS.
- Conversion tool: ImageMagick `convert`, DDS output with `dds:compression=none`.
- Contact sheet: `docs/assets/006_independence_wave/decision_icons/contact_sheets/decision_icons_contact_sheet.png`
- Expected GFX file: `interface/006_independence_wave_icons.gfx`
