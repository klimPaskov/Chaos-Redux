# Event 006 Idea Icon Manifest

Status: regenerated set processed, converted, and wired; all final idea icons are 64x64 transparent DDS files.

This package adds static 64x64 idea/national-spirit icons for implemented Event 006 Independence Wave spirits. The source art is generated HOI4-style object-emblem art created after inspecting the repository idea and focus icon references. These icons are non-flag assets, do not use Event 005 Soviet Collapse art, and avoid asserting real cultural or royal symbols for local-polity packages.

## Processing Notes

- Source mode: generated source art.
- Source sheets: `docs/assets/006_independence_wave/idea_icons/source_sheets/idea_icons_regen_sheet_*.png`
- Parent processing: generated source PNGs were sliced into individual source files, alpha-processed from chroma-key backgrounds, trimmed, resized, centered on 64x64 transparent canvases, and converted to DDS.
- Conversion tool: ImageMagick `convert`, DDS output with `dds:compression=none`.
- Contact sheet: `docs/assets/006_independence_wave/idea_icons/contact_sheets/idea_icons_contact_sheet.png`
- GFX file: `interface/006_independence_wave_icons.gfx`
- Gameplay file: `common/ideas/006_independence_wave_ideas.txt`
- Validation: final DDS dimensions are 64x64 with alpha; unused canvas pixels are transparent.
