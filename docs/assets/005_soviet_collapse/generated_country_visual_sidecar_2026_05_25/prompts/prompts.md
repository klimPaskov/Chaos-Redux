# Event 005 Generated Country Visual Prompts

Scope: fictional/custom Event 005 Soviet Collapse country visuals, generated with Codex `$imagegen` on 2026-05-25.

## Shared flag constraints

Each prompt requested a 2x2 sheet of four horizontal flag panels for communism, democratic, fascism, and neutrality variants. The constraints were:

- clean intentional HOI4-readable flag art
- 1936-1945 period heraldry tone
- no readable text, letters, numbers, or watermarks
- no real national flag copying
- no simple tricolors
- no recolor-only variants, flipped duplicates, or copied single-emblem variants
- readable at 82x52, 41x26, and 10x7
- no modern props, modern signage, UI artifacts, people, flagpoles, or photorealistic waving cloth
