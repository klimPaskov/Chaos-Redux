# Dhyana Seal Source Prompts

Generation mode: `$imagegen`.

Final source method:
- One painterly seed frame of a carved lotus shrine medallion was generated first to replace the old geometry-heavy seal direction.
- Direct edit attempts drifted too hard, so the final animation source used one generated `4x3` animation-source grid with the same shrine medallion repeated across twelve subtle breath states.
- The grid source is saved as `docs/assets/003_holy_realm_buddhahood/animations/holy_realm_dhyana_seal/notes/holy_realm_dhyana_seal_source_grid.png`, then sliced into frame PNGs under `source_frames/`.

Shared prompt requirements:
- Centered blue-violet lotus meditation seal for The Holy Realm.
- Carved shrine-medallion look with worn enamel petals, aged gold trim, dark lacquered disc, and pale lavender inner flame-orb.
- Square icon composition suitable for a `96x96` decision icon crop.
- No character, face, person, readable text, letters, numbers, flags, maps, weapons, blood, gore, monsters, or modern UI.
- No simple concentric line-art mandala, flat vector logo look, or placeholder geometry.
