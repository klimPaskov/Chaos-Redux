# Buddha Mandate Portrait Frame Plan

The loop uses eight independent generated frames. Motion must come from separate source artwork, not from transform-only processing of one still image.

| Frame | State | Notes |
| --- | --- | --- |
| 0 | Resting radiance | Calm golden portrait, halo low, crown jewels quiet. |
| 1 | First breath | Halo brightens behind the crown; cheeks and brow gain warmer light. |
| 2 | Crown ignition | Top jewel and side ornaments glint; eyes remain calm. |
| 3 | Awakened eyes | Eye light increases; forehead jewel glows. |
| 4 | Full mandate | Halo reaches peak density; gold aura strongest but face remains readable. |
| 5 | Compassionate settling | Aura softens; mouth and lower ornaments regain shadow detail. |
| 6 | Quiet jewel echo | Crown glints recede; background becomes paler. |
| 7 | Return to rest | Halo dims to frame 0 intensity while preserving alignment. |

## Processing

- Resize/crop each source to `156x210`, centered on the face and crown.
- Assemble horizontally into `portrait_THR_buddha_mandate_sheet.png`.
- Convert the sheet to `gfx/leaders/THR/portrait_THR_buddha_mandate_animated.dds`.
- Build a contact sheet and GIF preview for review.
