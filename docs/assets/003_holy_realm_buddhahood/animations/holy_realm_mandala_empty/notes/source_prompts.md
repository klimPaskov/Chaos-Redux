# Holy Realm Empty Seat Mandala Source Prompts

Source mode: `$imagegen` built-in tool.

All frames used the same production constraints:

- Square game UI source frame for a Hearts of Iron IV scripted GUI state.
- One centered circular Holy Realm Mandala with stable camera and stable scale.
- Dark charcoal background, ashen grey rings, muted grey-violet haze, vacant centre.
- No readable text, letters, numbers, glyph inscriptions, UI labels, watermark, faces, people, Buddha figure, flags, weapons, maps, extra symbols outside the Mandala, red crisis palette, gold teaching palette, flames, active lamp, or glowing golden teaching light.
