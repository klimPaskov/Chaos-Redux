# Holy Realm Awakened Mandala Source Prompts

All frames used built-in `$imagegen` source mode. Frame 000 established the seed composition; frames 001 through 011 reused the visible seed mandala as the consistency reference.

## Common Prompt Body

```text
Use case: stylized-concept
Asset type: HOI4 scripted GUI mandala animation source frame, final crop will be 420x420
Primary request: Create a frame of a 12-frame loop for an awakened Holy Realm mandala. Keep the same centered sacred wheel and lotus mandala, same camera/scale/framing/center anchor, pale ivory core disk, warm gold concentric rings, dark charcoal field, subtle Himalayan silhouette along the lower background edge. Painterly symbolic WWII-era strategy game UI art, readable at 420x420 and when scaled to half size.
Composition/framing: perfectly centered, symmetrical, square, no border, no labels.
Lighting/mood: serene sacred gold light, restrained contrast, atmospheric but crisp.
Constraints: no readable text, no letters, no numerals, no faces or face-like shapes, no maps, no modern UI, no watermark.
```
