# Holy Realm Dormant Mandala Frame Plan

All frames are generated as independent source art and processed into a single horizontal frame sheet. The sequence loops back to frame 000 without a visible jump.

| Frame | Beat | Visual Notes |
| --- | --- | --- |
| 000 | Sealed rest | Static-like teal-grey mandala, center closed and dim. |
| 001 | First breath | Outer ring brightens very slightly in cold teal. |
| 002 | Inner listening | Star lines become marginally clearer without gold activation. |
| 003 | Locked pulse | Center shows a muted blue-green pulse, still sealed. |
| 004 | Quiet peak | Outer ring and axes reach their softest readable glow. |
| 005 | Fading seal | Glow recedes from outer ring toward the center. |
| 006 | Stone stillness | Lines return to low contrast and center darkens. |
| 007 | Loop seam | Nearly matches frame 000 with only a faint residual teal edge. |
